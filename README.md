# 🍚 ontime-call（饭点提醒）

> 还在因为下班慢了 1 分钟被同事抢电梯、错过午饭窗口、加班到忘了晚饭而懊悔吗？

**ontime-call** 就是为这种"就差一分钟"的日常痛点写的。它安静地待在系统托盘里，到点了准时弹窗提醒你——带贪睡按钮，忙完再看也来得及。

---

## 🎯 它能解决什么问题

| 痛点 | 怎么解决 |
|------|----------|
| 🏃 下班慢 1 分钟，电梯被抢光 | 提前提醒，准时收工 |
| 🍜 午饭窗口人太多，错过就得等到 1 点 | 11:27 准时弹，先去排队 |
| 🌙 加班到忘了时间，晚饭也没吃 | 17:37 晚饭提醒 + 20:57 加班结束提醒 |
| 📅 周末不需要加班提醒，但工作日要 | 工作制设置：双休/单休/大小周，自动跳过 |
| 🎮 打着游戏/写着代码不想被打断 | 专注勿扰：检测到你还在动键盘就不弹，等你静止了再提醒 |
| 🔔 偶尔需要一次性提醒（比如 30 分钟后开会） | 单次提醒，顺延或固定时刻任选 |
| 🎨 默认界面太无聊 | 6 套主题 + 6 个自定义图标随便换 |

## 🚀 快速开始

### 方式一：双击启动（推荐）

解压后直接双击 `meal-reminder.vbs`，程序会缩到系统托盘静默运行。

### 方式二：命令行

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -STA -File ".\meal-reminder.ps1"
```

### 方式三：设为开机自启

打开主界面 → 设置 → 勾选「开机自动启动」。不需要管理员权限，会在启动文件夹里放一个快捷方式。程序自带的 `meal-reminder.vbs` 就是为此准备的——设置开机自启时会自动用 `wscript.exe` 调用它。如果你重命名或移动了 VBS 文件，开机自启可能失效，需要重新设置。

## 📋 完整功能列表

### 定时提醒
- 🥡 **午饭提醒** — 默认 11:27，可自定义时间
- 🍲 **晚饭提醒** — 默认 17:37，可自定义时间
- 🏠 **加班结束** — 默认 20:57，可自定义时间
- ⏰ **自定义提醒** — 无限条，每条可单独设置：时间、标题、提醒文字、是否强提醒、是否播放声音
- 🔂 **单次提醒** — 支持"按当前时间顺延"（如 30 分钟后）和"固定时刻"（如今晚 9 点）两种模式

### 提醒行为
- 💤 **贪睡** — 弹窗提供 5/10/30 分钟贪睡按钮，忙完再看
- 🔊 **强提醒** — 红色标题 + 双次响铃，重要提醒不会错过
- 🧘 **专注勿扰** — 检测键盘鼠标空闲状态，你还在操作就不弹，等静止 60 秒再提醒（最多延迟 N 分钟强制弹出）
- 🪟 **Windows 原生通知** — Win10/11 上使用原生 Toast，出现在右下角不抢焦点；旧系统自动回退到 Form 弹窗

### 场景模式
- ✈️ **出差模式** — 一键暂停所有提醒，回来再开
- ⏸️ **今日暂停** — 今天就别烦我了，明天自动恢复
- 🚫 **单独屏蔽** — 不想被加班提醒烦？单独关掉它就行

### 工作制
- 📅 **双休** — 周六周日自动跳过提醒
- 📅 **单休** — 可选周日或周六休息
- 📅 **大小周** — 隔周双休，支持设置某周一为大小周锚点

### 界面美化
- 🎨 **6 套主题** — 浅色 / 深色 / 库洛米 / 皮卡丘 / 线条小狗 / 猪猪侠
- 🖼️ **自定义图标** — 6 个内置图标可选，或用你自己的 .ico 文件
- 📊 **吃饭统计** — 托盘提示文字显示"🍚 本周准点吃饭 5/7 天"，带周报弹窗

### 其他
- 📌 **系统托盘常驻** — 左键单击托盘图标显示主界面，右键展开菜单
- 🔒 **单实例运行** — 重复启动会唤醒已有窗口，不会开两个
- 📦 **便携打包** — 运行 `打包便携版.ps1` 一键生成 ZIP

## 📖 使用说明

### 主界面

打开主界面可以看到：
- 顶部**实时时钟**
- **当前状态**卡片：当前模式、各提醒开启情况、专注勿扰状态
- 每条提醒的开关按钮（屏蔽/开启）
- 底部操作按钮：公司上班 / 出差中 / 测试弹窗 / 自定义提醒 / 设置 / 今日暂停 / 退出

### 托盘菜单

右键点击系统托盘图标：
- **显示主界面** — 把窗口叫回来
- **公司上班 / 出差中** — 快速切换模式
- **今日暂停提醒** — 暂停今天所有提醒
- **自定义提醒** — 打开自定义提醒管理
- **设置** — 打开设置面板
- **退出** — 完全退出程序

### 设置面板

打开设置可以调整：
- 开机自动启动
- 界面模式（6 套主题切换）
- 应用图标（跟随主题 / 内置图标 / 自定义 .ico 文件）
- 弹窗播放声音
- 饭点强提醒
- 专注勿扰（键鼠静止秒数 + 最大延迟分钟数）
- 工作制（双休 / 单休 / 大小周）
- 每日提醒时间微调（午饭/晚饭/加班结束可各自修改时间）

### 自定义提醒

点击「自定义提醒」按钮打开管理窗口：
- 左侧列表显示所有自定义提醒及其状态（待提醒 / 今日已提醒 / 已关闭）
- 右侧编辑：时间（时/分）、标题、提醒内容
- 可单独控制每条提醒的启用/禁用、声音开关、强提醒
- 支持新增、保存、删除

## 📁 配置与数据

所有运行时数据存放在用户本地应用数据目录，与程序目录分离：

```text
%LOCALAPPDATA%\ontime-call\
├── config\meal-reminder.config.json   # 配置（主题、提醒时间、工作制等）
├── logs\meal-reminder.log             # 运行日志
├── state\meal-reminder.show           # 单实例唤醒标记
└── cache\themes\                      # 主题图片缓存（自动生成）
```

升级时直接替换程序目录即可，配置和日志不会丢失。

## 🛠️ 技术栈

- **语言**：PowerShell 5.1+
- **GUI**：Windows Forms（.NET Framework 原生）
- **通知**：WinRT COM `Windows.UI.Notifications`（零依赖原生 Toast，自动回退 Form 弹窗）
- **平台**：Windows 10 / Windows 11
- **权限**：不需要管理员权限
